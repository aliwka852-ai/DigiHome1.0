# DigiHome 1.0 — Визуальный концепт приложения

## 🎨 Дизайн-система

### Цветовая палитра
```
Основные цвета:
- Primary Blue: #2196F3 (управление, активные элементы)
- Dark Bg: #0F1419 (тёмный фон для контрастности)
- Light Gray: #E8E8E8 (вторичный фон)
- Success Green: #4CAF50 (активные устройства)
- Warning Orange: #FF9800 (предупреждения)
- Accent Purple: #9C27B0 (специальные функции)
- Text Dark: #1A1A1A (основной текст)
- Text Light: #7A7A7A (вторичный текст)
```

### Типография
- **Заголовки (H1)**: 32px, Bold, Primary Blue
- **Подзаголовки (H2)**: 24px, SemiBold, Text Dark
- **Основной текст**: 16px, Regular, Text Dark
- **Вторичный текст**: 14px, Regular, Text Light
- **Метки**: 12px, Medium, Text Light

### Иконография
- Стиль: Material Design Icons (округлые формы)
- Размеры: 24px, 32px, 48px, 64px
- Цвета: Primary Blue, Success Green для активных состояний

## 🚀 Взаимодействие (Interactions)

### Swipe gestures
- Свайп влево/вправо: переключение между комнатами
- Свайп вверх: открыть меню
- Свайп вниз: обновить данные

### Long Press
- На устройство: открыть расширенное меню
- На сценарий: редактировать/удалить

### Double Tap
- На toggle: быстрое включение/выключение
- На комнату: открыть детали
